# GEO 发布前必检 Checklist

> 评分前的快速自检——10 项基础 + 5 项加分。

## 基础检查（10 项）

- [ ] 标题包含核心实体，15-30 字
- [ ] 首屏摘要 80-120 字 + 2-4 个 Bullets 要点
- [ ] 数据密度 ≥ 每 300 字 1 个具体数据
- [ ] Bullets 占比 ≥ 60%
- [ ] 键值对 ≥ 6 组
- [ ] 实体命名全文统一（品牌 / 产品 / 术语）
- [ ] 对比区（如有对比场景）使用表格，客观中立
- [ ] FAQ 5-8 条，每条 50-80 字完整回答
- [ ] EEAT 信号齐全：经验 + 专业 + 权威 + 可信
- [ ] Schema 标记已部署（Product / Article / FAQPage）

## 加分项（每项可抵扣主维度扣分）

- [ ] 独家数据/调研（+5）
- [ ] 可视化图表/流程图（+3）
- [ ] 引用权威来源 ≥ 3 个（+3）
- [ ] 作者资质说明（+2）
- [ ] 更新日期标注（+2）

## 评级速查

| 总分 | 结论 | 行动 |
|:----:|:-----|:-----|
| ≥ 80 | 通过 | 可发布，建议补加分项再涨分 |
| 70-79 | 修改后通过 | 24h 内针对扣分项优化重提 |
| < 70 | 不通过 | 退回重写，先做 ERE 建模再起稿 |
